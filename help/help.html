Below, we have captured a number of questions that people have asked about this site. The questions are divided into three categories, General, OS/2, and Windows.

General

  • Bamba Streaming Audio Information
    This course employs Bamba streaming audio technology that runs best under Netscape 4.0, Internet Explorer 4.0 or later versions. The Bamba player is a Java applet. If you don't have a Java-enabled browser or if Java is disabled in your browser, you won't hear audio. Older browsers don't handle the Bamba player effectively. You will experience palette shifts when audio is streaming. Also, older hardware will not have the proper audio card to handle streaming audio.
  • No Plug-in to Set Up
    To hear audio, just give the Bamba player time to download fully. The download is automatic... you don't have to do anything or find any plug-ins! When the download is finished, you'll hear audio for the discussion item you're on. Once the Bamba player has downloaded, you can use the player controls to stop the audio, then resume it at the beginning or anywhere in the audio file.

    If your audio is not working properly or if you prefer to read, we've provided all the lecture text in a scrolling box on the right of the screen. It's your choice. Get the information any way it suits you best.

  • More Courses
    Four courses are being developed in this series of "Intro to…" lectures. All the courses use the Dr Java Bean character and run on a 640X480 or higher screen resolution.
  • Font Size
    We've noticed that some machines running 800X600 make the text on our site very difficult to read, especially if your default font is smaller than 12pt. To fix this problem, change your default font to 12pt following the steps below.
    1. Select Options from your browser pulldown menu
    2. Select General Preferences
    3. Select the Fonts tab
    4. Where it says "Use the Proportional Font", select Choose Font
    5. Select a 12pt default font size

    • OS/2 Users
  • Audio Problems
    If you can't hear audio when you take the course, the problem can come from several places. Some things you can do to correct the problem are:
    1. Make sure your audio volume controls are set properly.
    2. Make sure you have Multimedia capabilities on your machine.
    3. Are you running a Netscape 2.02e browser as a minimum?

    • Windows 95 Users

  • Audio Problems for ThinkPad Users
    There's a known problem with some Thinkpad models running Mwave audio. The audio plays but it breaks up while it's playing, Try installing the patch for MWAVE.
    1. Right click here and pick "Save Link As" or "Save Target As".
    2. Browse to c:\temp. Click Save.
    3. Under your Windows95 Start Menu, click "Run".
    4. Type c:\temp\patch.exe Click on "OK".
    5. The program will launch and walk you through installing the patch.
